  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • organizationId: required(number)

    The organization the location belongs to

    Example:

    12345
  • locationName: required(string)

    The display name of the location

    Example:

    Bank Street

    Example:

    500 Bank Street

    Ottawa
